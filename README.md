# Сайт для организации мероприятий 

## Стек технологий

- **Frontend:** HTML, CSS, JavaScript, Bootstrap (или Tailwind CSS, если выбран)
- **Backend:** Node.js, Express.js
- **Database:** SQLite

## Запуск проекта (ЛОКАЛЬНО)

1.  **Установи зависимость: NodeJS (Мне пришлось качать с оффициального сайта, прежде чем писать эту команду в терминал)**
    ```bash (ну или любой другой терминал)
    npm install
    ```
    *(Эта команда прочитает файл `package.json` и установит все перечисленные в `dependencies` и `devDependencies` библиотеки)*

2.  **Запусти сервер:**
    ```bash (ну или любой другой терминал)
    npm start
    ```

После запуска сервер будет доступен по адресу `http://localhost:3000`.
